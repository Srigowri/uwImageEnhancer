Reproducing results:

To use the webapp locally using our saved model Use UnderWaterApp.zip

1. Unzip UnderwaterApp
2. Open terminal from that folder and type command 
   flask run
3. Go to the url: http://127.0.0.1:5000/
4. You can select any underwater test images from the image folder
5. Click on enhance

-------------------------------------------------------------

Metrics

SSIM,PSNR and MSE metrics for the output image can be calculated using the Metrics folder and running the python code. But, this requires changing the file name within the metrics.py file in Metrics folder.
